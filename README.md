# Clash Royale — Texnologiya Reportu

Bu report Clash Royale oyununun texnologiyasını araşdırır.
İçərisində oyunun real həyatda necə işlədiyini göstərən kod nümunələri var.

## Fayllar

**game_packet.cpp** — Oyun zamanı client ilə server arasında gedən paketlərin C++ strukturu

**battle_server.erl** — Erlang dilində yazılmış real-time döyüş serverinin məntiqi

**supercell_auth.js** — Supercell ID sistemi ilə istifadəçi girişinin JavaScript kodu

**social_client.js** — Dost əlavəsi, klan və chat üçün WebSocket əlaqəsi

**scylladb_events.py** — Oyun hadisələrinin ScyllaDB verilənlər bazasına yazılması

**clash_royale_api.py** — Rəsmi Clash Royale API vasitəsilə oyunçu məlumatlarının çəkilməsi
